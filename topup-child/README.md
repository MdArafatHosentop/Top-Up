# Free Fire Diamond Top-up WooCommerce Child Theme

এটি একটি সিম্পল এবং কার্যকরী ওয়ার্ডপ্রেস চাইল্ড থিম যা WooCommerce প্রোডাক্ট পেজে **Player ID** ফিল্ড যোগ করে। এটি মূলত ফ্রি ফায়ার ডায়মন্ড টপ-আপ ওয়েবসাইটের জন্য তৈরি।

## বৈশিষ্ট্যসমূহ:
- ✅ প্রোডাক্ট পেজে কাস্টম Player ID ইনপুট।
- ✅ ভ্যালিডেশন সিস্টেম (আইডি না দিলে অর্ডার হবে না)।
- ✅ চেকআউট এবং অর্ডারে প্লেয়ার আইডি প্রদর্শন।
- ✅ অ্যাডমিন প্যানেলে অর্ডারের সাথে আইডি দেখার সুবিধা।

## কিভাবে ব্যবহার করবেন:
1. এই রিপোজিটরি থেকে জিপ ফাইলটি ডাউনলোড করুন।
2. আপনার WordPress ড্যাশবোর্ডে গিয়ে `Appearance > Themes > Add New`-এ যান।
3. ফাইলটি আপলোড করে **Activate** করুন।
4. নিশ্চিত করুন যে আপনার মেইন থিম (যেমন: Hello Elementor) ইনস্টল করা আছে।
